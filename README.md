# 🛡️ 坦克大战

一个经典的双人/单机坦克对战游戏，使用 HTML5 Canvas 和原生 JavaScript 开发。

![坦克大战](./tank-game.gif)

## 🎮 游戏特点

- 🌟 经典坦克对战玩法
- 🎨 精美的视觉效果
- 💥 粒子爆炸效果
- 📈 难度随分数提升
- ⚔️ 支持 AI 和多人对战

## 🎯 游戏规则

作为玩家控制的蓝色坦克，击败红色敌方坦克！
- 生命值：100 点
- 每次击杀：+10 分
- 等级提升：每 100 分升一级

## ⌨️ 操作说明

| 按键 | 功能 |
|------|------|
| **W** | 向上移动 |
| **S** | 向下移动 |
| **A** | 向左移动 |
| **D** | 向右移动 |
| **↑** | 向左转 |
| **↓** | 向右转 |
| **空格** | 发射炮弹 |
| **P** | 暂停游戏 |
| **R** | 重置游戏 |

## 🚀 快速开始

### 方法一：直接在浏览器打开

1. 下载或克隆此仓库
2. 用浏览器打开 `index.html` 文件
3. 点击"开始游戏"按钮

### 方法二：本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后访问 `http://localhost:8000`

- CSS3 动画
